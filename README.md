# Library Book CRUD Manager

A simple Python program for managing books in a library with CRUD operations.

## What is CRUD

CRUD stands for:
- **C**reate: Add new books
- **R**ead: View all books
- **U**pdate: Modify existing books
- **D**elete: Remove books

## What This Program Does

This program helps you manage a book collection. You can perform these actions:

- Add new books to the library
- See all books in the library
- Change book details like title and author
- Remove books from the library
- See the total number of books

## How to Use

1. Run the program
2. You will see a menu with these choices:
   - 1. Add Book - Add a new book with unique ID
   - 2. View Books - Show all books in a list
   - 3. Update Book - Change book information
   - 4. Delete Book - Remove a book
   - 5. Exit - Close the program

3. Type the number of your choice and press Enter

## Book Information

Each book stores three pieces of information:
- Book ID: A unique code for each book
- Title: The name of the book
- Author: The writer of the book

## Important Features

- Prevents duplicate book IDs
- Shows total book count on main menu
- Clear and simple menu navigation
- Confirmation messages for all actions
- Error messages when books are not found

## Requirements

- Python 3 installed on your computer

## How to Run

1. Save the code as a Python file
2. Open command prompt or terminal
3. Go to the folder where you saved the file
4. Type: python code.py
5. Press Enter to start

## What Happens to Data

- All book information is stored in memory
- Data is lost when you close the program
- No files are created or saved
